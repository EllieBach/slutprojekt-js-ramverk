 # About
 This project is a library search:
 * search for a book
 * favorite a book
 * write a review
 * set rating
 * set pages read
 * see how many books you read
 * book cover
   
## Technologies used:
* React 
*  Redux
*  Sass
* routing 

## Installation
1. clone the repository:
   git clone https://github.com/your-username/your-repository.git
2. Navigate to the project directory:
   cd project-directory
3. npm install
